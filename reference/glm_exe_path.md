# Path to the installed GLM executable

Returns the expected path of the GLM binary in the per-user data
directory. The file may not exist yet if
[`glm_install()`](https://flare-forecast.github.io/GLMAEDr/reference/glm_install.md)
has not been run.

## Usage

``` r
glm_exe_path()
```

## Value

Character string.
